## 四、Webserver V0.3：用户 BST 索引（下午）

### 设计思路

在 V0.2 链表基础上叠加 BST 索引. 索引结点存 `user_t *` 指针而非拷贝, 内存由链表统一管理, 避免重复 free.

### 新增文件

| 文件 | 说明 |
|------|------|
| `include/user_index.h` | `user_index_node_t` (left/right/*user), `user_index_t` (root/size), 接口 |
| `src/user_index.c` | BST 构建, 中序输出, 查找, compare (链表 vs 树步数对比), 释放 |
| `data/users_large.csv` | 20 条用户数据 |
| `test/test_day03.sh` | 自动化测试 6 项 |

### 修改文件

| 文件 | 改动 |
|------|------|
| `src/main.c` | 新增 `index`, `find-index`, `compare` 子命令; 新增 `#include <stdlib.h>` |
| `Makefile` | 新增 `user_index.o`, `test03` 目标 |
