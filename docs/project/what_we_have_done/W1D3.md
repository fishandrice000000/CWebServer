# W1D3：C语言知识强化：数组与结构 — 上午总结

## 一、课程内容

- 二叉树定义、性质、满二叉树与完全二叉树
- 二叉树的顺序存储与链式存储
- 二叉排序树 (BST)：查找、新增、创建、删除（叶子/单子树/双子树）
- 中序遍历（有序输出）

## 二、训练一：BST 查找与新增

**路径:** `training/w1d3/train1/`

| 功能 | 说明 |
|------|------|
| 加载 | 从 `contacts.csv` 读取 name / phone, 以 name 为 key 插入 BST |
| 查找 | 递归 BST 查找, 输出找到的联系人或 "Not found" |
| 新增 | 创建新结点并插入到正确位置, 同名跳过 |
| 遍历 | 中序遍历输出所有联系人 (按键升序) |

## 三、训练二：BST 删除、修改、保存

**路径:** `training/w1d3/train2/`

| 功能 | 说明 |
|------|------|
| 删除 | 支持三种情况: 叶子直接删, 单子树替换, 双子树用中序后继替换 |
| 修改 | 查找后修改 phone 字段 |
| 保存 | 中序遍历写入 CSV |
