# W1D2：C语言知识强化：指针与链表 — 上午总结

## 一、课程内容

- 动态内存分配：malloc / free / calloc / realloc
- 单链表操作：初始化、判空、销毁、求长、头插法、尾插法、按值查找、按位插入、按位删除
- 文件操作：fopen / fclose / fgets / fputs / fread / fwrite / fscanf / fprintf / 随机读写
- CSV 文件格式与读写

## 二、课堂训练

### 训练一：CSV + 链表 CRUD

**路径：** `training/w1d2/train1/`

| 功能 | 说明                                                    |
| ---- | ------------------------------------------------------- |
| 加载 | 从 `users.csv` 读取用户信息, 跳过标题行, 尾插法建立链表 |
| 遍历 | 显示全部用户的 id / username / password / contact       |
| 增加 | 交互录入用户信息, 追加到链表尾部                        |
| 删除 | 按 id 删除指定用户                                      |
| 修改 | 按 id 查找, 修改 username / password / contact          |
| 查找 | 按 id 查找并显示用户信息                                |
| 保存 | 将链表数据写回 CSV                                      |

**构建：** `make && ./user_mgr`

### 训练二：有序链表

**路径：** `training/w1d2/train2/`

| 功能     | 说明                                    |
| -------- | --------------------------------------- |
| 有序加载 | 从乱序 CSV 读取, 按 id 递增有序插入     |
| 有序插入 | 新增用户时找到正确位置插入, 检测重复 id |
| 排序     | 按 username 递增冒泡排序                |
| 增删改查 | 与训练一类似, 但插入保持有序            |
| 保存     | 写回 CSV                                |

**构建：** `make && ./ordered_list`
