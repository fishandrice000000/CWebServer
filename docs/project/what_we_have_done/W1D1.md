## 二、课堂训练

### 训练一：单向链表

**路径：** `training/w1d1/linked_list/`

**内容：** 实现 `stuInfo` / `ElemType` / `ListNode` 数据结构，编写创建链表、头插法、尾插法、遍历链表等基本算法。

**构建与运行：**
```bash
cd training/w1d1/linked_list
make && ./linked_list
```

**输出：**
```
=== Linked List Traversal ===
Name: Charlie     Age: 19
Name: Alice       Age: 20
Name: Bob         Age: 21
```

### 训练二：文本格式化程序 fmt

**路径：** `training/w1d1/fmt/`

**模块划分：**
| 文件 | 职责 |
|------|------|
| `word.c` / `word.h` | `read_word` — 从输入读取一个单词 |
| `line.c` / `line.h` | 行缓冲区：添加单词、判断满、输出（含两端对齐） |
| `fmt.c` | main 函数，主循环逻辑 |

**构建与运行：**
```bash
cd training/w1d1/fmt
make
echo "The Very Hungry Caterpillar ..." | ./fmt
```

## 三、Webserver V0.1 工程

### 项目结构

```
CWebServer/
├── include/
│   ├── config.h            # 配置结构体与接口
│   ├── log.h               # 日志接口
│   └── http_response.h     # HTTP 响应构建接口
├── src/
│   ├── main.c              # 入口：加载配置 → 初始化日志 → 输出 HTTP 响应
│   ├── config.c            # 配置解析（load_config / print_config）
│   ├── log.c               # 日志实现（文件写入）
│   └── http_response.c     # 构建 HTTP/1.1 200 响应
├── config/
│   └── server.conf         # 服务器配置文件
├── www/
│   └── index.html          # 静态首页
├── test/
│   └── test_day01.sh       # Day01 自动化测试脚本
├── build/                  # 构建产物（gitignore）
├── logs/                   # 日志输出（gitignore）
├── docs/project/
│   ├── debug-log.txt       # 调试记录
│   └── what_we_have_done/  # 各课时完成总结
├── Makefile                # 构建系统
└── .gitignore
```

### Makefile 目标

| 命令 | 功能 |
|------|------|
| `make` | 编译项目，产物输出到 `build/` |
| `make run` | 编译并运行（使用 `config/server.conf`） |
| `make test` | 运行自动化测试脚本 |
| `make clean` | 清理所有构建产物 |
